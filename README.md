# Estudando-HTML-e-CSS

Este projeto é um exercício feito para praticar Html e Css com os conhecimentos que adquiri até o momento. Consiste em uma página simples de informações de um jogo eletrônico,  especificamente Resident Evil:  Requiem. Todas as informações retiradas são encontradas em sites de fácil acesso como www.ResidentEvil.com ou simplismente a Wikipedia, afinal o objetivo do projeto é apenas treinar Html e Css.
Houve a prática de escolha de paleta de cores utilizando a ferramenta Adobe Color, para fazer o gradiente do site, treinando harmonia de cores de acordo com o estilo do jogo escolhido.

Este projeto foi desenvolvido exclusivamente para fins educacionais e de portfólio. Todos os direitos intelectuais, marcas registradas e ativos (como imagens, personagens e nomes) do jogo Resident Evil: Requiem pertencem à CAPCOM.
